1.01 - Computational Thinking

An algorithm is a set of instructions used to solve a problem or complete a task.

This could include baking a cake, navigating from one place to another, or constructing flat-pack furniture. Algorithms do not have to be written in code. However, algorithms are usually written in pseudocode or flowcharts.

Example 1 shows an algorithm written in pseudocode.

Example 1
total ← 0
FOR num ← 1 TO 100
  total ← total + num
ENDFOR
OUTPUT total

What does the code in Example 1 do?

Hover/tap to reveal Sums all the numbers from 1 to 100 and outputs the result.
